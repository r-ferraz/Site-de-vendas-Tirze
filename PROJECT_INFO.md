# Akin Project Context

## Overview
This folder contains the rebranded Akin website, cloned from VoySaude.

## Branding
- **Name**: Akin (Never use "Voy")
- **Palette**: Dark Blue (#0047AB), Light Blue (#E3F2FD), White.

## Key Files
- `index.html`: Main landing page.
- `questionario.html`: 25-step health assessment.
- `pagamento.html`: Checkout page with ViaCEP integration.

## Persistence
To keep this project updated on GitHub, any AI agent MUST follow the directive in `../directives/akin_site_update.md` and use the script in `../execution/git_sync.py`.
